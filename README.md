# Password Generator
## Homework Assignment Three

This is my third homework assignment. It is a password generator.

### Cool things it does:
#### It is customizable. 

You can determine:
  - How many characters you want it to be.

You can pick whether you want a combo of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
  
#### All if the push of one button!
